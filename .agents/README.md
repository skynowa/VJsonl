# Codex Project Workspace

This directory contains project-local AI extensions for VJsonl.

Suggested structure:

- `skills/` - reusable task instructions in Codex skill format.
- `agents/` - role descriptions for specialized project agents.
- `prompts/` - reusable prompts/checklists/snippets.

These files are intentionally project-local, so they can evolve with the codebase.
